How to enable dark mode on Raspberry Pi OS

Rachit Thakker

May 12, 2022

Posted in:

#Raspberry Pi#Linux

The developers in the Raspberry Pi and Linux community have created many different operating systems for the SBC. It is easy to switch to dark mode via a graphical tool in most of them, including the Raspberry Pi OS. The only difference between the other OSs and Raspberry Pi OS is that the latter doesn't have the tool in the menu right out of the box.

This guide will help you to switch to a dark theme for the official OS from the Raspberry Pi foundation.

Using the Terminal

Open the terminal and use the following command to launch the LXAppearance tool.

1 lxappearance

This above image shows the default look of the tool. Under Widget tab, Adwaita is the default option.

To enable dark mode just choose Adwaita-dark and click on Apply. You can close this tool now.

That's it! Your OS should now be easier on your eyes although the icons in the Taskbar might appear a little different, but it won't break the system.

Adding the LXAppearance tool in the Main Menu

If you do not want to use the terminal every time you want to open this tool, you can add it to the Main Menu.

Open the Menu by clicking on the Raspberry Pi icon in the taskbar. Click on Preferences -> Main Menu Editor.

In the editor, click on Preferences under Applications on the left. Mark the checkbox for Theme and Appearance Settings. Click on OK. Now you should see this option available under Preferences in the Main Menu.
